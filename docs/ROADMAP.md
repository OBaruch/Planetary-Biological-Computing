# Roadmap

## Phase 1: Robust Local Simulator

- Harden FastAPI runner lifecycle.
- Improve status/config/session observability.
- Expand event system and frontend controls.
- Keep simulator/fallback language honest.

## Phase 2: Recording Replay

- Add replay support for real or shared recordings if available.
- Compare sessions through exported JSONL/CSV summaries.
- Add charts for spike rate, entropy, stress, recovery, and resilience.

## Phase 3: Controlled Cortical Cloud Session

- Implement a dedicated CL1/Cortical Cloud adapter.
- Map stimulation intents to reviewed stimulation plans.
- Add safety gates, rate limits, and manual approval controls.

## Phase 4: Public Visualization

- Prepare a deployable frontend/backend demo.
- Add curated narrative scenes and presenter mode.
- Add recording/export for video demos.

## Phase 5: Open Dataset And Analysis

- Publish synthetic session datasets.
- Document schemas and analysis notebooks.
- Add reproducible benchmark scenarios.

## Phase 6: Paper / Demo / Installation

- Prepare a scientific-artistic writeup.
- Build an installation-grade planet visualization.
- Document ethical boundaries and future biological deployment requirements.
